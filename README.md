# 🍅 Python CLI Pomodoro Timer & Study Logger

A custom command-line interface (CLI) Pomodoro timer built entirely in Python. I designed this tool to help me maintain deep focus during long study sessions and track my daily productivity.

## ✨ Key Features
* **Custom Timers:** Dynamically input your preferred focus and break intervals (e.g., 50/10 or 25/5).
* **Marathon Mode:** Set a target number of sessions, and the program runs a continuous loop.
* **Real-Time UI:** Clean terminal countdown using the `time` module and carriage returns for a live clock effect.
* **Auto-Logging:** Automatically records every completed session (timestamp, duration, and subject) into a `study_history.txt` file.

## 🚀 How to Run
1. Clone this repository to your local machine.
2. Open your terminal in the project folder.
3. Run the script:
   ```bash
   python pomodoro.py
